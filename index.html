Problem Statement

Develop an end-to-end system for code-switched Egyptian Arabic-English speech that transcribes to mixed text then converts Arabic portions to Arabizi (Franco-Arab Latin script). Targets daily conversational code-switching common in youth texting.

[file:30]

Proposed Pipeline

Speech Input → Whisper ASR → Code-Switched Text
Code-Switched Text → NileChat → Arabizi Output

Example: "I acquired a lot of knowledge ya3ny 3an el networks" [file:30]

Datasets

  • ArzEn-ST: 12h code-switched Egyptian Arabic-English speech (6,216 utterances) [file:30]
  • EgyptianBench: 2K paired Arabic-Arabizi rows [file:30]
  • EgyptianHellaSwag: 40K parallel Arabic-Arabizi text pairs [file:30]
